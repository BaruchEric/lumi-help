# Sharing with family

Every item in Lumina is **Private** by default — only you see it. You can change that:

- **Shared** — household members can see it. Pick from your Family page.
- **Public** — anyone with the link can see it. Great for sharing a recipe publicly.

Look for the 🔒 / 👪 / 🌐 icon on recipes and logs to see each item's current state.
